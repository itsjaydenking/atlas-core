# atlas-core

Atlas is a relational worldbuilding platform prototype focused on structured entities and relationships.

## Tech Stack

- React + Vite + JavaScript (JSX)
- React Router
- ESLint
- Prettier
- Standard CSS (no Tailwind)

## Run Locally

```bash
npm install
npm run dev
```

## Quality Checks

```bash
npm run lint
npm run build
npm run format:check
```

## Current Scope (V0)

- Desktop-style application shell with header, sidebar, main content, and inspector panel
- Route-based pages for dashboard, entities, relationships, timeline, rules, and settings
- In-memory mock data for entities and relationships
- Dashboard stats derived from mock data
- Interactive entity list and details view
